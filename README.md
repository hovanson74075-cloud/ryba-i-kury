# Рыба и Куры — GitHub Pages + WhatsApp

Статический сайт для GitHub Pages с корзиной и оформлением через WhatsApp.

## Быстрый старт
1. Создайте репозиторий и загрузите файлы из архива.
2. В `settings.js` укажите свой номер: `window.WHATSAPP_NUMBER = '79001234567'` (формат 7XXXXXXXXXX).
3. Включите GitHub Pages: Settings → Pages → Deploy from a branch.
4. Откройте сайт и проверьте оформление заказа — откроется чат WhatsApp с подставленным текстом.

## Каталог
- Товары в `products.json`.
- Для офлайн‑редактирования откройте `admin-lite.html`, поправьте, нажмите «Скачать products.json» и замените файл в репозитории.
